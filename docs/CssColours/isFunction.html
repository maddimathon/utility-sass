    Function isFunction

    • isFunction(
          value: string,
          args?: { roundingFactor?: number; strict?: boolean },
      ): boolean

      Whether the given string matches one of the regular expressions in CssColours.Regex (representing css colour functions).

      Parameters

      • value: string
      • Optionalargs: { roundingFactor?: number; strict?: boolean }

      Returns boolean

      See

      CssColours.parseFunction — Used to check for valid formats.

      Since

      0.1.0-beta.0.draft

      Format Examples

      These formats are tested with CssColours.parseFunction.
