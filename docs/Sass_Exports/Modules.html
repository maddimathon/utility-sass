    Sass (scss) Modules

    These scss "modules" are meant as extensions of the built-in sass modules.

    Generally, you won’t want to @forward these modules, unless you would also forward the built-in module it is meant to replace.

    For example:

    @use 'pkg:@maddimathon/utility-sass/list'; // adds extras to 'sass:list'

    list.flatten(...); // added

    list.index(...); // built-in
    list.nth(...); // built-in
