    Namespace CssColours

    Utilities for css colour values.

    0.1.0-beta.0.draft

    A set used to check if a string is a Keyword type.

    systemColors

    A set used to check if a string is a SystemColor type.

    Functions

    isFunction

    Whether the given string matches one of the regular expressions in CssColours.Regex (representing css colour functions).

    isKeyword

    Whether this is one of the keywords in CssColours.Keyword (i.e., valid css values but that do not represent a specific colour [e.g., 'transparent']).

    isSlug

    Whether this is one of the keywords in CssColours.Slug (i.e., keywords used for defined web-safe colours).

    isSystemColor

    Whether this is one of the system colors in CssColours.SystemColor (i.e., forced-colors mode keywords).

    parseFunction

    Parses a string and returns data for valid CSS colour codes.
